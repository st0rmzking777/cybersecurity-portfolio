## Independent Projects

### Project Kingfisher — End-to-End Phishing Simulation & Detection Lab
*April 2026 – current*

A three-phase independent lab project simulating real-world phishing attacks and building defender telemetry to detect them. Designed to bridge offensive and defensive perspectives.

**Phase 1 — Attacker Infrastructure (complete):**
- Three-VM lab architecture on VMware Workstation Pro
- Two Gophish phishing campaigns simulating credential harvesting
- MITRE ATT&CK mapping: T1566.002, T1204.001, T1056.003, T1078

**Phase 2 — Defender Telemetry (complete):**
- Sysmon deployment for endpoint visibility
- PowerShell Script Block Logging
- Windows process creation and authentication event auditing

**Phase 3 — SIEM & Detection Engineering (in progress):**
- Wazuh SIEM deployment
- Custom detection rules and alert correlation
- Mapping detections back to MITRE ATT&CK techniques

See the `project-kingfisher/` folder for documentation, methodology, and detection logic.

*May 2026*

Volunteered informal security advisory for a small business. Identified critical gaps including weak password hygiene, absence of MFA, and no Zero Trust policy. Performed threat modelling using MITRE ATT&CK framework and recommended Bitwarden, Brave browser, and third-party penetration testing prior to internal application deployment.

Key outcome: declined an informal request for unauthorised testing and escalated through proper authorisation channels; demonstrating professional ethics under social pressure.
